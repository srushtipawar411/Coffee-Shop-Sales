☕ Coffee Shop Sales Analysis Dashboard (SQL + Power BI)
📌 Project Overview
The Coffee Shop Sales Analysis Dashboard is a data analytics project designed to analyze and visualize sales performance of a coffee shop using SQL and Power BI. The goal of this project is to transform raw transactional data into meaningful insights that can help businesses make data-driven decisions.
This project analyzes sales data to understand customer purchasing behavior, product performance, store performance, and sales trends. An interactive Power BI dashboard is created to visualize key metrics and business insights.
The dashboard allows users to easily track performance indicators such as total sales, total orders, quantity sold, product category performance, store location performance, and hourly sales patterns.

🎯 Business Problem
- Coffee shops generate a large amount of transactional data daily. Without proper analysis, it becomes difficult for business owners to understand:
- Which products generate the most revenue
- Which store location performs the best
- When customers purchase the most products
- How sales vary across days and hours
- Which product categories drive the highest sales
- This project solves this problem by building an interactive analytics dashboard that provides clear insights into sales performance.

🎯 Project Objectives
- The main objectives of this project are:
- Analyze overall sales performance
- Identify top-selling products
- Analyze sales by product category
- Compare sales performance across store locations
- Analyze weekday vs weekend sales trends
- Identify peak sales hours
- Build an interactive Power BI dashboard for business insights

📂 Dataset Description
The dataset used in this project contains transactional records from a coffee shop. Each row represents a single sales transaction.
- Important Fields in the Dataset
- Column Name	Description
- Transaction ID	Unique identifier for each transaction
- Transaction Date	Date when the sale occurred
- Transaction Time	Time when the sale occurred
- Store Location	Location of the coffee shop branch
- Product Category	Category of the product
- Product Type	Specific product purchased
- Quantity	Number of items purchased
- Unit Price	Price per item
- Total Sales	Total revenue generated from the transaction
- The dataset helps analyze customer behavior, product demand, and sales performance.

📈 Sales Analysis
1️⃣ Sales by Store Location
- This visualization compares sales across different coffee shop branches.
- Locations included in the dataset:
  Hell's Kitchen
  Astoria
  Lower Manhattan
 This helps identify which location contributes the most to overall revenue.

2️⃣ Sales by Product Category
- Products are categorized into different groups such as:
  Coffee
  Tea
  Bakery
  Drinking Chocolate
  Coffee Beans
  Flavours
  Packaged Chocolate
 The analysis shows which categories generate the highest revenue.

3️⃣ Top Selling Products
- The dashboard highlights the most popular products purchased by customers.
- Examples of top-performing products:
  Barista Espresso
  Brewed Chai Tea
  Hot Chocolate
  Gourmet Brewed Coffee
  Brewed Herbal Tea
 This insight helps businesses understand customer preferences.

4️⃣ Weekday vs Weekend Sales

- The dashboard compares sales generated during:
  Weekdays
  Weekends
 This helps businesses understand when customer demand is highest and allows better workforce and inventory planning.

5️⃣ Sales Trend Over Time
- A time-series visualization shows how sales change over the selected period.
- This analysis helps identify:
  High-performing days
  Sales fluctuations
  Overall business growth trends

6️⃣ Hourly Sales Analysis
- This section analyzes sales by hour of the day.
- The analysis helps identify peak hours when customers purchase the most products, which is useful for staffing and operational planning.
- Typically, coffee shops experience peak demand during morning hours when customers buy coffee before work.
